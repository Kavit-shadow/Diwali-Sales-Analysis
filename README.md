# 🪔 Diwali Sales Analytics Dashboard (Advanced Streamlit Business Intelligence App)

## 📌 Project Overview

This project transforms traditional Diwali sales analysis into a **next-generation interactive Streamlit Business Intelligence Dashboard** built for real-time data exploration, executive reporting, and advanced visual analytics.

Using **Python, Plotly, and Streamlit**, the raw Diwali sales dataset is cleaned, analyzed, and presented through a **production-ready analytics dashboard** featuring dynamic filters, advanced charts, KPI cards, exportable reports, and business insights.

---

## 🚀 Live Application Features

### ✨ Advanced Dashboard Capabilities

* Interactive **Executive KPI Cards**
* Dynamic **Sidebar Filtering System**
* Multi-Tab Professional Dashboard Layout
* Advanced **Plotly Visualizations**
* **CSV / Excel Report Export**
* AI-Style **Business Recommendations**
* Responsive **Dark-Themed Modern UI**
* Production-Ready **Streamlit Architecture**

---

## 🎯 Project Objectives

* Analyze customer purchasing behavior during festive seasons
* Discover high-performing states, demographics, and product categories
* Build a modern BI dashboard for real-time business insights

## 📈 Key Business Insights

* Married women aged **26–35** are the highest spending segment
* Top revenue is concentrated in a few high-performing states
* Certain product categories dominate festive purchases
* Occupation and geography strongly influence spending behavior
